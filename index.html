De que trata el proyecto?

Este trabajo trata de horganizar los elementos necesarios para iniciar y completar una RAID en el videojugo Destiny 2. A la hora de plantear el proyecto justo estaba muy enganchado a este mismo juego, debido a esto decidi intentar realizar una Base de Datos que pudiera servir para poder tener en cuenta todo lo necesario para completar una Raid.

Que se tiene que tener en cuenta?

Lo primero que se tiene que tener en cuenta a la hora de entrar a una Raid es tu nivel y el de tu escuadra, no hay que olvidar que las Raids son el contenido End Game de Destiny 2, y pese a que no lo parezca estamos jugando a un RPG por lo tanto intentar entrar sin el nivel adecuado es directamente una perdida de tiempo. Ademas de esto tambien se tienen que tener en cuenta los contratos que llevas para maximizar las recompensas y casi lo mas importante el equipamiento, sobretodo las ARMAS y ARMADURAS EXCEPCIONALES, ya que hay piezas concretas que nos facilitaran mucho completar los objetivos de esta. Sobretodo tenemos que tener mucho cuidado debido a que dentro de la Raid ya no se puede cabiar el equipamiento hasta que se complete.
